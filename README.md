# Hepsiburada Mobil Android Bootcamp Final Project

## Tasks: Receiving, processing and searching data from the Itunes API

## Key features
 
 * ✔ MVVM
 * ✔ Retrofit
 * ✔ Navigation Component
 * ✔ View-Binding
 * ✔ Pagination
 * ✔ Coroutines
 * ✔ Detail Page
 * ✔ RecylerView
 * ✔ Searching
